# Sacca Car Beauty Figma Handoff

## Design System

Colors:
- Main background: `#FFFFFF`
- Section alternate: `#F7F8FA`
- Light app background: `#F5F7F8`
- Card: `#FFFFFF`
- Primary text: `#111111`
- Secondary text: `#666666`
- Secondary border: `#E5E7EB`
- Lime accent: `#8DFF2F`
- Lime hover: `#74D61C`
- Lime gradient end: `#00C853`

Typography:
- Headings: Space Grotesk
- Body: Inter
- Hero: 56-72px, 800 weight, line-height 0.98
- Section heading: 32-48px, 800 weight
- Body: 14-18px, 400-500 weight
- Labels: 12px, 700 weight, uppercase, 0.18em tracking

Spacing:
- Base unit: 8px
- Page gutters: 16px mobile, 24px tablet, 32px desktop
- Section vertical padding: 64-80px
- Card radius: 16px
- Soft shadow: `0px 10px 30px rgba(0,0,0,0.06)`
- CTA gradient: `linear-gradient(135deg, #8DFF2F 0%, #00C853 100%)`

Core Components:
- Sticky navbar: logo left, navigation center, search/wishlist/cart/account/WhatsApp right
- Primary button: lime fill, dark text, subtle lift on hover
- Secondary button: white fill, gray border, dark hover border
- Outline button: dark border, dark fill on hover
- Product card: square image, discount badges, wishlist icon, hover image zoom, add-to-cart reveal
- Mobile bottom bar: fixed cart and WhatsApp actions
- Form fields: white fill, gray border, lime focus ring
- Skeleton loader: soft gray shimmer

Screens To Build In Figma:
- Home: hero, vehicle finder, categories, featured products, testimonials, WhatsApp CTA
- Product listing: left filters, toolbar with sort, responsive product grid
- Product detail: gallery, details, quantity, add-to-cart, WhatsApp order, trust badges
- Cart: item list, quantity controls, order summary, WhatsApp checkout
- Checkout: customer details, order summary, pre-filled WhatsApp flow
- About: brand story, values, operating principles
- Contact: WhatsApp CTA, contact cards, inquiry form

Responsive Frames:
- Desktop: 1440px
- Tablet: 768px
- Mobile: 375px with 2-column product grid and sticky bottom bar
